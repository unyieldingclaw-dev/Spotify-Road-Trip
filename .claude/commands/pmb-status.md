# /pmb-status

Run `mb status` and present the output.

Do not add commentary, interpretation, or suggestions unless an Attention item requires it. If the output shows `0 Issues`, respond with the output only. If the output shows Attention items, list them clearly and offer the most direct remediation command for each (e.g., `mb audit`, `mb init`, `mb upgrade`).
